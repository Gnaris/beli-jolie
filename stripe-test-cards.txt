  Pour tester le virement bancaire :
  1. Creer un PaymentIntent avec payment_method_types: ["customer_balance"]
  2. Dans le dashboard Stripe (mode test), aller dans le PaymentIntent
  3. Cliquer "Fund" pour simuler la reception du virement

  Ou via CLI :
  stripe trigger customer.cash_balance.funds_available
